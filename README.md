# 📅 Interactive Calendar Web App

A fully functional **interactive calendar web application** built using **HTML, CSS, and JavaScript**.

This project allows users to:
- Select multiple dates (click + drag)
- Add events to selected dates
- View events with different colors
- See visually merged event bars
- Delete events by clicking on them

---

## 🚀 Features

### ✅ Calendar Functions
- Real month navigation (Next / Previous)
- Dynamic calendar generation
- Drag-to-select multiple dates

### 📝 Event Management
- Add events to multiple selected dates
- Multiple events per day supported
- Click event to delete it

### 🎨 UI Features
- Different colors for different events
- Smart color consistency (same event = same color)
- Merged event bars (horizontal continuity)

### ⚡ Behavior
- No backend required
- No `localStorage` used
- Events reset on refresh (temporary session-based)
